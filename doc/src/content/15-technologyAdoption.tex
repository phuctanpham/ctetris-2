  \hline
  \textbf{GitHub Gist}     & REST API v3    & Content delivery channel: \texttt{manifest.json} chứa SHA per chapter; SHA-diff $\rightarrow$ chỉ fetch chapter có nội dung mới; không full-replace DB. \\

  \textbf{Cloudflare Worker}& V3            & REST proxy: \texttt{POST /api/records} nhận game record JSON $\rightarrow$ ghi vào D1; \texttt{GET /api/records} cho Board popup. \\
  \hline
  \textbf{Cloudflare D1}   & V3 (edge SQLite)& Global leaderboard; schema \texttt{sync\_Records}; 1-way sync: local SQLite là source of truth, D1 là read-only replica cho public board. \\
  \hline
  \textbf{Cloudflare DO}   & V3 (Durable Objects)& Rate limiting + OTP session cho auth flow; không persist game state. \\
  \hline
\end{tabularx}

  \hline

  \hline

  \hline
  \texttt{test-build.yml}     & push nhánh khác\newline path: \texttt{app/**} & Build native + WASM trên 3 OS song song & matrix:\newline windows / ubuntu / macos \\
